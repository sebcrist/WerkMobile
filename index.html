How Werkmobile Works

The system is designed to be modular. You configure categories and data mappings so a photo can be attached to the exact record your team cares about, whether that is a work order, shipment, invoice, or another custom document type. The current setup is intended for teams already using a Microsoft SQL Server database which Werkmobile integrates with.

  1. Capture: Take or upload a photo from mobile.
  2. Link: Associate it with the right document in your SQL-backed workflow.
  3. Enrich: Run custom SQL queries to automatically save extra context with the image.

Teams then use the dashboard to filter, review, and organize images by category, user, or operation. This gives production, shipping, and support teams a shared visual record that is easy to search and audit. Support for other relational databases should also be possible with integration adjustments; please contact me if you need that setup.

If you do not need tracking and simply want to transfer pictures from your phone to a specific folder on your server, there is also a transfer-only mode in the app.

Visit the information page on the app (top right) to learn how to set up your own local API server and connect with your existing systems!
